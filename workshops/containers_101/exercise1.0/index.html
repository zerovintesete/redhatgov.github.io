Install Docker

Install Docker
$ sudo -i
# yum install -y docker

Start Docker

To start the docker binary and have it managed by systemd we first need to enable it and tell systemd that we want it to run a boot time. Then to start the docker daemon right away we use the start command.

systemctl enable docker
systemctl start docker
systemctl status docker
