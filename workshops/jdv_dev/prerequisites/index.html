Prerequisites for JDV Development

Return to Workshop

  • Next: Exercise 1 - Install JDV and JBDS →

What you need

  • A free account on Red Hat Developers to download JBoss Data Virtualization and JBoss Developer Studio
  • A Java Development Kit (JDK) 8
  • git available as a local command
  • Node.js runtime, 6.x or higher
  • Postman for testing
  • An accessible PostgreSQL instance
    • Recommended: Download a pre-built VM, ISO or docker image from TurnKey Linux. The provided VM/ISO also provides an web interface for loading data. Step by step instructions are provided for a Turnkey linux instance
